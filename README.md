<h3>Description</h3>

ScatterTV is a Flask based Web-Application that plots TV Shows on a CanvasJS graph in accordance to the shows ratings using weekly updated datasets that get pulled directly 
from IMDB's website! 
<br>

<h3>Live Site</h3>

www.scattertv.com - Deployed using Microsoft Azure
<h3>Requirements</h3>
Flask==2.0.1
